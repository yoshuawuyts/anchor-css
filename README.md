# anchor-css
A tiny css ([styl -w](https://github.com/visionmedia/styl#styl1)) framework

## Installation
With [bower](http://bower.io/) installed:
````bower install anchor-css````

## Directory structure
````
.
├── utilities			// helper classes
│   ├── base
│   ├── colors 
│   ├── grid          
│   └── media
├── micro				// atoms
│   ├── animations
│   ├── buttons
│   ├── images
│   ├── navigation
│   ├── progress
│   ├── tables
│   └── typography
└── macro				// widgets
    ├── dialogs                 
    └── feedback       
````
